# 🕵️ Mini Mystery

Mini Mystery is a detective investigation game built using HTML, CSS, JavaScript, Node.js, and Express.js. Players investigate different cases, collect evidence, question suspects, and identify the criminal.

## Features

- 3 different mystery cases
- Evidence collection system
- Suspect questioning
- Investigation report
- Case solved screen with crime timeline

## Mystery Cases

1. 💎 The Black Diamond
2. 🖼️ The Missing Painting
3. 💻 Office Data Theft

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Local Storage
